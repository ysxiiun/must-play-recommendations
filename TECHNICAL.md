## 项目概述

### 简介

**Must-Play Recommendations** 是一个基于 Claude Code 的 Agent Skill，为用户提供城市必玩景点的智能推荐服务。通过融合 FlyAI CLI 多数据源，实现5A级景区全覆盖 + 热门好评景点推荐，以友好的 Markdown 格式输出详细推荐结果。

### 核心能力

| 能力 | 说明 |
|-----|------|
| 5A级景区全输出 | 获取城市所有国家5A级景区 |
| 热门好评推荐 | ai-search 获取人气景点 |
| 智能补充 | 不足5个时用4A级景区补充 |
| 信息丰富 | 景区描述、游玩时长、交通指南 |
| 门票信息整合 | 免费/收费/具体价格 |

### 适用场景

- 旅行规划前期的景点调研
- 城市旅游攻略制作
- 目的地景点快速了解
- 行程安排参考

```

    │
    ▼

│ • 5A级景区优先保留完整信息                                   │
│ • 热门好评补充描述、时长、交通                                │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ Step 4: 智能补充（可选）                                      │
│ 如果 5A + 热门好评 < 5:                                      │

└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ Step 5: 整合输出                                             │
│ • Markdown 格式化                                            │
│ • 门票信息整合（免费/收费/价格未知）                          │
│ • 标签生成（基于类别映射）                                    │
│ • 分组展示（5A级景区 / 热门好评推荐）                         │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
```

### 数据结构

#### 5A级景区数据结构（来自 search-poi）

```typescript
interface POIAttraction {
  name: string;           // 景点名称
  address: string;        // 地址
  category: string;       // 类别
  poiLevel: string;       // 等级 (5A/4A/3A)
  freePoiStatus: 'FREE' | 'NOT_FREE' | 'UNKNOWN';  // 是否免费
  ticketInfo?: {          // 门票信息
    ticketName: string;   // 门票名称
    price?: string;       // 价格
    priceDate?: string;   // 价格日期
  } | null;
  jumpUrl: string;        // 预订链接
  mainPic: string;        // 主图链接
}
```

#### 热门好评数据结构（来自 ai-search 文本解析）

```typescript
interface AISearchAttraction {
  name: string;           // 景点名称
  description: string;    // 景区描述/特色亮点
  duration: string;       // 游玩时长建议
  address: string;        // 地址
  traffic: string;        // 交通指南
  jumpUrl: string;        // 预订链接
}
```

#### 融合后输出数据结构

```typescript
interface MergedAttraction {
  name: string;           // 景点名称
  level: string;          // 等级 (5A/4A/热门好评)
  category: string;       // 类别
  address: string;        // 地址
  ticket: string;         // 门票信息（整合后）
  description: string;    // 景区描述
  duration: string;       // 游玩时长
  traffic: string;        // 交通指南
  tags: string[];         // 标签列表
  jumpUrl: string;        // 预订链接
  mainPic: string;        // 主图链接
}
```

### 门票信息整合规则

```typescript
function formatTicket(status: string, ticketInfo: any): string {
  switch (status) {
    case 'FREE':
      return '免费开放';
    case 'NOT_FREE':
      if (ticketInfo?.ticketName) {
        return `收费（${ticketInfo.ticketName}）`;
      }
      return '收费（具体价格请查询）';
    case 'UNKNOWN':
    default:
      return '价格请以景区为准';
  }
}
```

## API 参考

### FlyAI CLI 命令

#### search-poi

获取指定城市的景点列表。

```bash
flyai search-poi --city-name "{城市}" --poi-level {等级}
```

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `--city-name` | 是 | 城市名称 |
| `--poi-level` | 否 | 景点等级（1-5），5为最高 |

**返回字段**：

| 字段 | 说明 |
|-----|------|
| `name` | 景点名称 |
| `address` | 地址 |
| `category` | 类别 |
| `poiLevel` | 等级 |
| `freePoiStatus` | 是否免费 |
| `ticketInfo` | 门票信息 |
| `jumpUrl` | 预订链接 |
| `mainPic` | 主图 |

#### ai-search

AI 语义搜索，获取热门推荐。

```bash
flyai ai-search --query "{查询语句}"
```

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `--query` | 是 | 自然语言查询 |

**返回格式**：文本（Markdown）

**包含信息**：
- 景点名称
- 特色描述
- 游玩时长建议
- 地址/交通
- 预订链接

## 常见问题

### Q1: 为什么有些景点没有图片？

**A**: 图片来源于 FlyAI 数据，部分景点可能缺少图片数据。可以尝试使用 ai-search 获取的描述信息。

### Q2: 门票价格为什么显示"未知"或"具体价格请查询"？

**A**: 门票价格可能有以下情况：
- `freePoiStatus = "NOT_FREE"` 且 `ticketInfo` 为空
- 价格数据未实时更新
- 建议点击预订链接查看实时价格

### Q3: 为什么有些城市5A级景区很少？

**A**: 5A级景区是国家旅游局评定的最高等级景区，数量有限。Skill 会自动用4A级景区或热门好评景点补充。

### Q4: 如何添加自定义触发词？

**A**: 修改 `SKILL.md` 文件的 YAML frontmatter 中的 `triggers` 列表：

```yaml
triggers:
  - 有什么好玩
  - 景点推荐
  - 你的自定义触发词
```

### Q5: 数据更新频率是多少？

**A**: 数据来源于 FlyAI CLI 的实时查询，每次调用都会获取最新数据。
