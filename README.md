# Must-Play Recommendations

> 城市必玩景点推荐 Agent Skill

## 简介

本技能为用户提供城市必玩景点的智能推荐服务。融合5A级景区和热门好评景点，通过 FlyAI CLI 多数据源获取景点信息，以友好的 Markdown 格式输出详细推荐结果。

## 核心能力

- **5A级景区全输出**：获取城市所有5A级景区
- **热门好评推荐**：ai-search 获取人气景点
- **智能补充**：不足5个时用4A级景区补充
- **信息丰富**：景区描述、游玩时长、交通指南
- **门票信息整合**：免费/收费/具体价格

## 推荐逻辑

1. **获取5A级景区** - 使用 `search-poi` 获取全部5A级景区
2. **获取热门好评** - 使用 `ai-search` 获取人气景点
3. **数据融合去重** - 合并数据，按名称去重
4. **智能补充** - 不足5个时用4A补充

## 数据来源

| API | 用途 |
|-----|------|
| `search-poi` | 获取5A/4A级景区基础信息 |
| `ai-search` | 获取热门好评景点、景区描述、交通指南 |
